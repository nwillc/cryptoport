# cryptoport
cli crypto portfolio 

Cli that produces output like:
```
        0.103451 BTC      3691.25
        7.030965 ETH     19281.38
              Total:     22972.63
```

## Use

1. Have your Nomics API key and holdings info ready. Setup your configuration:
```bash
$ cryptoport setup
```
2. Run...
```bash
$ cryptoport
```
